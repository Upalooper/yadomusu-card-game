# 宿の看板娘 返済記（ブラウザ版カード育成）

宿の看板娘をカードで育成し、18日以内に借金返済を目指すゲームです。  
`index.html` を開くとそのままプレイできます。

## 今回の改善ポイント
- **Slay the Spire風UI**
  - 画面下に手札を常時表示
  - 山札 / 捨て札 / 手札枚数 / 行動回数を常時表示
- **カード操作の改善**
  - カードにカーソルを合わせると右パネルに詳細表示
  - カードを上方向にドラッグすると使用
- **ゲーム進行**
  - 準備 → メイン → 営業終了の3フェイズ
  - 18日進行、返済処理、イベント回想、デッキ改造/圧縮、エンディング分岐

## 遊び方
1. 準備フェイズでランダム4択から1つ選ぶ
2. メインフェイズで手札カードを上にドラッグして使用
3. 行動0または「営業終了」でクローズ
4. 「次の日へ」で進行
5. 借金完済と高ステータスを目指す

## 拡張ポイント
- `CARD_LIBRARY`: 新カード追加
- `PREP_OPTIONS`: 準備行動追加
- `EVENTS`: 回想イベント追加
