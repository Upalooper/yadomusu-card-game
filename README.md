# 宿の看板娘 返済記（ブラウザ版プロトタイプ）

宿の看板娘をカードで育成し、18日以内に借金返済を目指す同人ゲーム向けプロトタイプです。  
`index.html` をブラウザで開くだけでプレイできます。

## 実装済み要素
- 1日を **準備 / メイン / 営業終了** の3フェイズで進行
- 準備フェイズでランダム4択（ショップ・アイテム・食事・休憩・交流）から1つ選択
- メインフェイズで手札4枚（効果により増減）を使い、行動回数を消費して育成
- 行動カードの例（給仕特訓、オーナーのアドバイス、やど娘のひらめき）を実装
- 営業終了フェイズで売上計算、借金返済、ランダム夜イベント
- カード強化/削除（デッキ改造）
- イベント回想ギャラリー（アンロック式）
- 18日経過または借金完済でエンディング分岐

## 遊び方
1. `index.html` を開く
2. 準備フェイズで1つ選ぶ
3. メインフェイズで手札をプレイ
4. 行動回数が尽きると営業終了
5. 「次の日へ」で進行
6. 借金を完済し、高ステータスを狙う

## 拡張しやすいポイント
- `CARD_LIBRARY` を増やせばカードを簡単追加可能
- `EVENTS` と `unlockEvent` 条件を拡張してイベントを増量可能
- エンディング条件を増やせばマルチエンド化可能
